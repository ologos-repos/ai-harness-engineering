## 15.1 Purpose

Configuration baseline governs the versioned definition of a harness deployment: models, instructions, prompts, tools, policies, memory schemas, and evaluations as one identifiable, restorable unit. Unengineered, the failure class is an undefined system: no two people can agree what version of the harness produced a given outcome.
