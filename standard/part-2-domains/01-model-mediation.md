## 1.1 Purpose

Model mediation governs how the harness abstracts model access: routing among models, substitution, version pinning, capability gating, and fallback on failure or degradation. Unengineered, the failure class is silent capability drift: a model change alters system behavior with no baseline, no qualification step, and no rollback path.
